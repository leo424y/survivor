# 什麼樣的產品可以成功？

一、

開發產品時，記得問三個問題。

1、它有用嗎？（Is it useful?）

2、它易用嗎？（Is it easy to use?）

3、它用起來令人愉悅嗎？（Is it delightful to use?）

二、

創業家孫陶然也說過類似的話。他說開發一個產品的時候，應該問自己五個問題。

1、給誰用？

2、他們用這個產品來解決什麼問題？

3、這個問題對他們而言有多重要？

4、我們的方法是否足夠簡單方便？

5、他們要付出的代價與所得是否匹配？

> “當我們對市場進展不夠滿意時，檢視這5個問題比檢視廣告更有效。”

我覺得這些問題太重要太正確了。
